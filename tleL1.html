LESSON 1: Introduction to CSS

Cascading Style Sheets (CSS) describes how HTML elements are to be displayed. It can control the layout of multiple webpages all at once. It is the language we use to style a webpage. CSS separates content from design, making websites easier to maintain and more attractive!

LAYERS OF A WEBPAGE
  • Content Layer (HTML) – It defines and lays out the text, images, videos, sounds, and other content elements you want to put in your webpage.
  • Presentation Layer (CSS) – It defines how people see your webpage in a browser.
  • Behavior Layer (JS) – It ensures interactivity and real-time user experience!
CAPABILITIES OF CSS
  • CSS makes your pages updateable.
  • CSS gives you control when placing objects exactly where you want them.
  • CSS allows you to position objects in 3 dimensions.
  • CSS allows you to create custom tags for specialized objectives.
ADVANTAGES OF USING CSS
  • It saves typing and development time.
  • Faster page loading.
  • Allows multiple link tags in one document.
  • Better design flexibility.
  • Easier to maintain and update.
  • Separates structure (HTML) from style (CSS).
